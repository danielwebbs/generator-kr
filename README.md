# Introduction
generator-kr is an opinionated knockout-Require front end project generator

# How to use
#### Generating Components

<code> yo kr:component </code>

#### Generating Services
<code> yo kr:service </code>

#### Updating app.js view integration
<code> yo kr:updateAppViews app </code>

### Prerequisites
* Npm

# Installation
To install Krns, clone the repository and then run
~~~bat
npm install
~~~
